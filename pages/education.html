My Web Development and Coding Journey:
This year, I was already thinking I should take coding classes and change careers.  Then I lost my job, when my employer filed bankruptcy and the corporate office was closed.  The time to begin my coding journey was now. I took the required assessment and I was accepted as a student, at Interface Web School, in the Foundations of Web Development course.  When I started as a student, at Interface, I had only completed a Udemy course: Everything You Need To Know Before You Code.  The Foundations of Web Development is a 10 week intensive course that teaches the skills needed to program and build websites from scratch.  I learned HTML5,  CSS3,  Bootstrap 4,  JavaScript  and  JQuery.

INTERFACE WEB SCHOOL:


I learned about color theory, user experience, user interface, and user centered design.   I was taught HTML tags, CSS Properties, and CSS Selectors.   I learned HTML/CSS Layouts and Structures, how to use/stylize images and Responsive Web Design.  I was introduced to JavaScript Usage, Data Types, Array, Variables, Scope, Functions, and Operators.  I learned how to create a parallax website, rotate objects, and create 3D cubes.  Sixteen Creative Agency class projects included a mobile project, a desktop project with media queries, a project that included mobile, desktop, + JQuery, as well as a Bootstrap project.
With no prior coding experience, the Foundations of Web Development course gave me the skills needed to design and build my first website using HTML/CSS and JavaScript.

UDEMY ONLINE COURSES:

Coding is all about practice, practice, practice. I quickly realized I needed to code 7 days a week.  I decided that I should also take online courses, that could reinforce what I was learning in class; yet, from a different perspective.  Most of my classes are through Udemy.  I have taken courses such as:  Learning HTML & CSS From Scratch,  Complete Bootstrap 4,  3 JavaScript Fundamentals courses, Web Development By Doing:  HTML/ CSS From Scratch, and Responsive Parallax HTML/CSS Web Design.
